git add .

---------------------------------------

git commit -m ""

---------------------------------------

отправка изменений на удаленный реп.:
git push -u origin master

где origin - это удаленный реп., а master - это локальный реп.

---------------------------------------

клонируем реп: 
git clone https://github.com/bz4work/spec3.git

---------------------------------------

получаем изменения с удаленного реп.: 
git pull origin master

где origin - это удаленный реп., а master - это локальный реп. 

---------------------------------------

просмотр удаленных репозиториев:
git remote -v

---------------------------------------

добавление удаленного репозитория:
git remote add имя_удал_реп https://github.com/bz4work/spec3.git

---------------------------------------

переименование удаленного реп.:
git remote rename pb paul

---------------------------------------

удаление удаленного реп.:
git remote rm paul

---------------------------------------
